# Java Ticket Booking System (CLI)

A command-line based ticket booking application developed using Core Java. The project demonstrates object-oriented programming concepts and basic business logic implementation.

## Overview
This application allows users to book and cancel tickets through a simple command-line interface. It manages seat availability, user details, and booking records entirely in memory.

## Features
- User registration
- Ticket booking
- Ticket cancellation
- Seat availability tracking
- Input validation and exception handling

## Technologies Used
- Java
- Core Java (OOP)
- Collections Framework
- Exception Handling
